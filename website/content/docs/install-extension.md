## VS Code Marketplace

1. **Browser** — Open the CoopAI listing on the [VS Code Marketplace](https://marketplace.visualstudio.com/) (or click Install on [coop-ai.dev](https://coop-ai.dev)).
2. **Browser** — Click **Install** — VS Code opens and installs the extension.
3. **Extension UI** — Reload VS Code if prompted. Open the Coop icon in the activity bar.

## Manual VSIX install

If your org received a `.vsix` file:

1. **Extension UI** — Command Palette → **Extensions: Install from VSIX…**
2. Select the `.vsix` file.
3. Reload VS Code.

## Verify installation

1. **Extension UI** — Activity bar shows the Coop icon.
2. Command Palette → **CoopAI Settings** opens the settings hub.
3. Command Palette → **CoopAI: Open Sidebar** opens the sidebar.

## Connect after install

Continue to [Getting started](/docs/getting-started) step 3 to sign in, or see [Extension settings](/docs/extension-settings) for full configuration.
